# Logos Directory

This directory contains logo images for the application.

## Gravyty Logo

To add the Gravyty logo:

1. Visit https://gravyty.com/
2. Right-click on the Gravyty logo in the header
3. Save the image as `gravyty-logo.svg` or `gravyty-logo.png`
4. Place it in this directory

The logo will be accessible at `/images/logos/gravyty-logo.svg` (or `.png`)

Preferred format: SVG for scalability, or PNG with transparent background.
